절대 경로(Absolute Path)#

현재 내위치가 어디든, 최상위 뿌리 (Root, /) 디렉토리 부터 시작하여 목표 폴더까지 모든 경로를 적는 방식

맨 앞에 루트를 의미하는 슬래시로 시작

$ cd Desktop/test/linux_pracitc

현재 폴더(.)나 상위 폴더(..) 기호를 적극 사용해야 함

../project

$ cd . 

$ cd ..

빈 파일 만들기#

touch [파일]

>는 덮어쓰기, >>는 내용 추가 가급적이면 > 사용 금지

$ touch hello.txt
echo "Hello, Linux!" > hello.txt
echo "두 번째 줄 추가" >> hello.txt

vi 편집기 익히기#

  • 기본문법
  • 개발자 필수 / 분석가 옵션
명령어상세 설명모드
i입력 모드 진입 (현재 커서 앞)명령 -> 입력
a현재 커서 다음 칸 진입 (현재 커서 앞)명령 -> 입력
ESC입력 모드 종료하고 종료 모드 복귀입력 -> 명령
:w변경 사항 저장(Write)ex 명령
:q편집기 종료(Quit)ex 명령
:wq저장 후 종료ex 명령
:q!저장 않고 종료ex 명령
u실행 취소(Undo)ex 명령
h,j,k,l좌, 하, 상, 우 이동이동
0현재 줄의 맨 처음으로 이동이동
$현재 줄의 맨 끝으로 이동이동
gg문서의 맨 첫 줄로 이동이동
G문서의 맨 마지막 줄로 이동이동
x커서 위치의 글자 1개 삭제삭제
dd커서가 있는 한 줄 삭제(잘라내기)삭제
yy커서가 있는 한 줄 복사복사

파일 폴더 복사, 이동, 삭제#

파일 복사#

cp [원본파일명][복사본파일명]

$ cp hello.txt hello2.txt
$ ls
    hello.txt  hello2.txt  memo.txt  src/

폴더 전체 복사#

cp -r [원본폴더명][복사본파일명]

$ cp -r linux_practice/ linux_practice_backup
$ls
    0528/  linux_practice/  linux_practice_backup/  linux_tutorial_01.md  README.md

파일 폴더 이동 / 이름 변경#

같은 위치에서 이름만 바꾸면 이름 변경 mv [파일명][파일명수정]

$ ls
    hello.txt  hello2.txt  memo.txt  src/
$ mv hello2.txt hello_copy.txt
$ ls
    hello.txt  hello_copy.txt  memo.txt  src/

파일명수정 대신 디렉토리경로 입력 시 이동 mv [파일명][디렉토리경로]

mission

  1. archive 폴더 생성
  2. hello_copy.txt 파일을 archive 이동 (mv 사용)
$ mkdir archive
$ ls
    archive/  hello.txt  hello_copy.txt  memo.txt  src/
$ mv hello_copy.txt ./archive/
$ ls
    archive/  hello.txt  memo.txt  src/
$ cd archive/
$ ls
    hello_copy.txt

파일 삭제#

rm [파일명]

$ ls
    hello_copy.txt, sample.txt
$ rm hello_copy.txt
$ ls
    sample.txt

폴더 삭제#

rm -r [폴더명]

$ ls
    archive/ hello.txt  memo.txt  src/
$ rm -r archive/
$ ls
    hello.txt  memo.txt  src/
